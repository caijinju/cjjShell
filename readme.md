# 版本0.1.0 -- 添加新版本的创建脚本

## 添加

### 脚本名 `newProject.sh`

在原版本上复制粘贴新的文件夹，并将项目所包含的版本号修改为新版本号

## 例子

当前已有版本V01

- 复制 V01 项目整个文件夹
- 黏贴并重命名为 V02 项目
- 修改相关文件夹和文件名
- 修改项目中 V01 的值

### 脚本使用

```
Administrator@USER-20170122UR MINGW64 ~/Desktop
$ sh a.sh
脚本使用：
用法：a.sh      旧项目名称      新项目名称
例如：a.sh      Test-V01        Test-V02

```

当前已存在新项目的文件夹时，会提示是否删除重建。输入Y/y 为确定删除，继续往下执行脚本；否则为取消删除，退出脚本。
